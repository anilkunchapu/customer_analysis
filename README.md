# Customer Behavior Analysis

## Project Overview

This project analyzes customer shopping behavior using Python, SQL, and Power BI.

## Tools Used

* Python (Data Cleaning using Pandas)
* SQL (Business Analysis)
* Power BI (Dashboard Visualization)

## Project Steps

1. Cleaned the raw dataset using Python.
2. Performed business analysis using SQL queries.
3. Built an interactive dashboard using Power BI.

## Key Insights

* Total Revenue: 233K
* Average Purchase Amount: 59.76
* Clothing category generates the highest revenue.
* Young adult customers contribute the most sales.

## Files in this Project

* `customer_behaviour_python.ipynb` → Data cleaning using Python
* `Customer Analysis SQL.sql` → SQL queries for business analysis
* `Customer Analysis DASHBOARD.pbix` → Power BI dashboard
* `customer_shopping_behavior.csv` → Dataset used
* `Screenshot.png` → Dashboard preview
